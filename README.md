# Boletín de Contrataciones del Estado Uruguayo

Sistema para visualizar y consultar contrataciones públicas del Estado uruguayo,
consumiendo datos de ACCE (Agencia de Compras y Contrataciones del Estado).

## Stack

| Capa       | Tecnología                        |
|------------|-----------------------------------|
| Backend    | NestJS + TypeScript               |
| Base datos | MongoDB                           |
| Frontend   | Angular (Standalone) + Material   |
